# SwiftGen 发布物

突发环境事件应急预案 AI 生成系统（SwiftGen）桌面端 dmg 公开发布仓库。

- `version.json` — 最新版本清单（应用内「检查更新」读取此文件比对版本）
- Releases — 各版本 dmg 安装包

> 代码仓库为私有，此 repo 仅托管公开发布物（版本号 + 安装包）。

## 安装
1. 从 [Releases](../../releases) 下载最新 `SwiftGen-x.y.z-arm64.dmg`
2. 双击 → 拖入 Applications → 首次**右键→打开**（无 Apple 签名，绕 Gatekeeper 一次）
